## Quick Start
### Getting the sources

```bash
git clone https://github.com/Samsung/iotjs.git
cd iotjs
```

### How to Build

```bash
tools/build.py
```

### How to Test

```bash
build/x86_64-linux/debug/bin/iotjs tools/check_test.js
```


For Additional information see [Getting Started](docs/Getting-Started.md).
